    What is DXNN®?

    DXNN is the all-in-one software platform that maximizes the performance of the DEEPX NPU.
    Through this platform, even AI models that have completed training on GPUs can be easily run at maximum speed on the DEEPX NPU.

    DXNN SDK Full Stack Architecture

    DXNN® (DEEPX Neural Network) SDK is a comprehensive software development environment that streamlines the entire AI deployment pipeline on DEEPX Neural Processing Units (NPUs). DXNN SDK integrates essential tools and frameworks for model compilation, optimization, simulation, and inference, all version-aligned to ensure seamless compatibility. While each tool can be installed separately, DEEPX offers a fully integrated ready-to-use environment optimized package for fast and efficient AI development and deployment, delivered as the DX-AS (AllSuite).

      DX-COM (Compiler)
      • DX-COM and DX-TRON in AI Model Compiler
      • Converts pre-trained ONNX models and JSON configurations into hardware-optimized .dxnn binaries.
      • Optimized for low-latency and high-efficiency inference.
      DX-Model Zoo
      • A curated collection of pre-trained and optimized neural network models.
      • Includes ONNX models, JSON files, and pre-compiled DXNN binaries.
      • Simplifies development and provides benchmark tools for performance comparison.
      Go to Link
      DX-Stream
      • A custom GStreamer plugin for real-time streaming data.
      • Enables seamless integration of streaming data into AI inference applications.
      • Tailored for vision AI use cases like video analytics and smart cameras.
      DX-APP
      • A sample application demonstrating how to run compiled models using DX-RT.
      • Includes ready-to-use code for common vision tasks.
      • Serves as a quick-start template for building custom AI applications.
      DX-RT (Runtime)
      • DX-RT and Device Driver for executing .dxnn models.
      • Directly interacts with DEEPX NPU hardware via a high-speed PCIe interface.
      • Provides C/C++ and Python APIs for flexible application control.
      Auto Install & Update (compiler, runtime, tools)
      • DX-AllSuite is a package that integrates and distributes the DXNN toolchain.
      • Automatically installs DXNN toolchain with synchronized version compatibility.
      • Install DXNN components easily and quickly with minimal commands.
